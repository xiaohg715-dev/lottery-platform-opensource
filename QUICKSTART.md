# Lottery Platform - 彩票平台开源项目

完整的项目结构和示例代码。

## 快速开始

### 后端服务
```bash
cd lottery-backend
mvn spring-boot:run
```

### 移动端APP
```bash
cd lottery-mobile-app
npm install
npm run dev
```

### 管理后台
```bash
cd lottery-admin-panel
npm install
npm run dev
```

### PC端门户
```bash
cd lottery-pc-portal
npm install
npm run dev
```

### 商户系统
```bash
cd lottery-shop-system
npm install
npm run dev
```

## 注意事项

⚠️ 本项目仅供学习研究使用,严禁用于商业运营或违法用途。

详细说明请查看各子项目的 README.md 文件。
