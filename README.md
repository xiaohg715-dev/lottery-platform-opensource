## 项目介绍

这是一套完整开源的体彩福彩竞技平台源码,包含体育竞技、福利彩票、实时直播等完整功能模块。

项目采用前后端分离架构,支持多端部署,所有源代码完全公开,可用于学习研究彩票算法、竞技赛事数据处理、实时直播技术等。

**本项目真正开源,不存在任何售卖行为。** 所有代码均可在项目仓库中获取。

## 功能模块

### 体育竞技
- 竞彩足球 - 胜平负、让球胜平负、比分、总进球、半全场
- 竞彩篮球 - 胜负、让分胜负、大小分、胜分差
- 北京单场 - 多种玩法组合
- 足球14场 - 传统足彩
- 任选九 - 九场胜负过关

### 福利彩票
- 双色球 - 6+1玩法
- 福彩3D - 三位数字竞猜
- 七乐彩 - 7+1玩法
- 快乐8 - 选号型彩票

### 数字彩票
- 排列3 - 三位数字
- 排列5 - 五位数字
- 大乐透 - 5+2玩法
- 七星彩 - 七位数字

### 直播功能
- 开奖动画直播 - 实时开奖号码滚动展示
- 赛事视频直播 - 足球篮球比赛实时视频流
- 弹幕互动 - 用户实时评论交流
- 多路流支持 - 同时观看多场比赛

## 技术架构

- JDK 17
- Spring Boot 3.x

- Redis 8.0
- WebSocket 实时通信

### 移动端 (uniapp-mayi-lottery)

- 支持编译到 Android/iOS/H5

## 核心算法

### 注数计算
实现了竞彩官网的标准注数计算算法,支持单关、过关等多种投注方式的准确计算。

### 奖金预测
根据赔率和投注金额,实时计算预测奖金范围,支持多串过关的奖金计算。

### 开奖算法
模拟真实开奖流程,包括随机数生成、号码验证、中奖判定等完整逻辑。

### 赔率计算
支持欧赔、亚盘、大小球等多种赔率体系的转换和计算。

## 部署说明

### 环境要求
- JDK 17+
- MySQL 8.0+
- Redis 8.0+
- Node.js 16+

### 快速启动

1. 导入数据库
```bash
mysql -u root -p < database/init.sql
```

2. 修改配置
```bash
# 编辑 application.yml
# 配置数据库连接、Redis地址等
```

3. 启动后端
```bash
cd mayi-lottery
mvn spring-boot:run
```

4. 启动前端
```bash
cd mayi-admin
npm install
npm run dev
```

详细部署文档请参考项目内的部署说明文件。
